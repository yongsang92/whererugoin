API 소개

서울 열린데이터 광장에서 상권배후지-추정유동인구 데이터를 csv파일, API를 통해 제공하고있습니다
하지만 csv파일은 너무 많은 정보로 인해 가독성이 떨어지고, API는 정보의 일부만을 제공합니다
이런까닭에 whereareyougoing은 핵심정보를 추출후 가공해서 제공합니다
또한 API 사용법을 익힐필요 없이 직관적인 사용법으로 API를 재구성했습니다
서울시에서 제공하는 원천데이터를 원하신다면 다음 링크로 가시면 됩니다 http://data.seoul.go.kr/dataList/OA-15582/S/1/datasetView.do
